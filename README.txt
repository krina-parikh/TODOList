Steps to run this application

- Option index.html from browser
- Add any task from text box
- Add button will add the item in the list
- User should get notification if he add same item
- Checkbox is to select the item from the list
- Selected item will be displayed as stroke letter 
- Bottom taskbar will show total number task and selected task
- Cross icon infront of item is for delete purpose
- Once user delete the iteam, he can see notfication on taskbar
